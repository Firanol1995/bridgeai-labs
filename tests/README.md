Tests
=

This folder is for automated tests (unit, integration, e2e). Add frameworks as needed:

- Unit tests: Jest / Vitest
- Integration: Playwright / Cypress

Suggested layout:

- tests/unit/  — unit tests
- tests/integration/ — integration tests
- tests/e2e/ — end-to-end tests

Run tests via `npm test` after adding your chosen test runner and configuring scripts.
